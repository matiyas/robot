# Support

If you need help using Robot Tank Control, there are several ways to get support:

## 1. Documentation

Check the `docs/` directory for detailed installation and troubleshooting guides:

- [DOCKER_SETUP_SUMMARY.md](docs/DOCKER_SETUP_SUMMARY.md)
- [COMPLETE_SETUP_GUIDE.md](docs/COMPLETE_SETUP_GUIDE.md)
- [GPIO_MONITORING.md](docs/GPIO_MONITORING.md)
- [VIDEO_STREAMING.md](docs/VIDEO_STREAMING.md)

## 2. Issues

Search for similar issues on the [Issue Tracker](https://github.com/matiyas/robot/issues). If you cannot find a solution, feel free to open a new issue. Please include as many details as possible, such as:

- Hardware details (Raspberry Pi model, motor drivers used, etc.)
- Software environment (Docker vs. Native)
- Steps to reproduce the issue
- Any error messages or logs you've gathered

## 3. Community

While we don't have a dedicated forum yet, we encourage discussion in the project's [Discussions](https://github.com/matiyas/robot/discussions) tab if it's enabled.

## 4. Troubleshooting Common Issues

See the **Troubleshooting** section in the [README.md](README.md#troubleshooting) for quick fixes to common hardware and software problems.

---

Thank you for using Robot Tank Control!
